# Streaming Service User Analysis

## Project Overview
This project analyzes user behavior, subscription patterns, revenue, engagement, retention, and regional preferences for a streaming service platform using Microsoft Excel.

The main objective is to identify business insights from customer data and present them through pivot tables, charts, dashboards, and a final presentation.

## Files Included
- `EXCEL SKILL TEST FINAL DELIVERABLES.xlsx` - Excel workbook containing data analysis, pivot tables, charts, and dashboard.
- `Streaming_Service_Final_Presentation . 1.pptx` - Final presentation summarizing the project insights and recommendations.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Dashboard Design
- PowerPoint

## Key Analysis Areas
- Subscription and revenue analysis
- User engagement analysis
- Demographic and behavioral analysis
- Retention and loyalty analysis
- Payment preferences and regional trends

## Key Insights
- Analyzed monthly revenue and subscription patterns.
- Compared user engagement using watch hours, movies watched, and series watched.
- Studied genre preferences across age groups.
- Reviewed active and inactive membership status.
- Identified payment method and country-wise user trends.

## Dashboard Features
- Pivot-based summary analysis
- Visual charts for business insights
- Interactive reporting structure
- Clean presentation-ready layout

## Business Recommendations
- Improve retention strategies for inactive users.
- Promote personalized recommendations to increase watch hours.
- Focus marketing campaigns based on country, language, and preferred genre.
- Use loyalty points and content downloads to improve customer engagement.

## Conclusion
This project demonstrates Excel-based business analytics skills by converting raw streaming service data into useful insights, dashboards, and presentation-ready recommendations.
